1. Information We Collect

1.1 Network Activity Data (Local Processing)

Our extension monitors network requests on websites you've explicitly approved for monitoring. This includes:

  • HTTP/HTTPS Requests: URLs, request methods, response status codes, and timing information
  • Error Information: Failed requests, timeout errors, and network connectivity issues
  • Response Metadata: Headers (excluding authorization tokens), content types, and response sizes

Important: We automatically exclude sensitive information including authorization tokens, cookies, passwords, and any data marked as sensitive by our privacy filters.

1.2 Extension Usage Data

  • Chat interactions and solution requests
  • Feature usage patterns and preferences
  • Error reports and diagnostic information
  • Extension settings and configuration choices

1.3 Optional Account Information

If you choose to create an account for cloud sync:

  • Email address for authentication
  • Profile preferences and settings
  • Solution history and bookmarks

3. Data Storage and Security

3.1 Local Storage (Primary)

Most of your data is stored locally in your browser using Chrome's extension storage APIs:

  • Network monitoring results and error logs
  • Solution cache and frequently accessed fixes
  • Extension settings and preferences
  • Recent chat history (last 50 conversations)

3.2 Cloud Storage (Optional)

If you enable cloud sync, we use Supabase (built on PostgreSQL) with the following security measures:

  • Encryption: All data encrypted in transit (TLS 1.3) and at rest (AES-256)
  • Access Controls: Row-level security ensures you can only access your own data
  • Infrastructure: Hosted on secure, SOC 2 compliant cloud infrastructure
  • Backups: Automated backups with point-in-time recovery

3.3 Data Minimization

We implement several privacy-by-design principles:

  • Automatic Sanitization: Remove sensitive headers and payloads before processing
  • Selective Monitoring: Only monitor domains you explicitly approve
  • Local Processing: Perform analysis locally when possible
  • Regular Cleanup: Automatically delete old logs and temporary data

5. Your Rights and Controls

5.1 Data Access and Portability

  • View Your Data: Access all stored data through the extension settings
  • Export Data: Download your chat history, bookmarks, and preferences
  • Data Portability: Export data in standard formats (JSON, CSV)

5.2 Data Control

  • Domain Management: Add or remove monitored domains at any time
  • Pause Monitoring: Temporarily disable monitoring without losing settings
  • Selective Deletion: Delete specific conversations or solution history
  • Complete Reset: Clear all local data and start fresh

5.3 Account Management

  • Account Deletion: Permanently delete your account and all associated data
  • Data Correction: Update or correct any stored information
  • Consent Withdrawal: Opt out of optional data processing at any time

5.4 GDPR Rights (EU Users)

If you're in the European Union, you have additional rights under GDPR:

  • Right to access your personal data
  • Right to rectification of inaccurate data
  • Right to erasure ("right to be forgotten")
  • Right to restrict processing
  • Right to data portability
  • Right to object to processing

6. Data Retention

6.1 Automatic Deletion

  • Network Logs: Deleted after 30 days
  • Chat History: Deleted after 90 days (unless bookmarked)
  • Error Reports: Deleted after resolution or 60 days
  • Temporary Data: Cleared on browser restart

6.2 User-Controlled Retention

  • Bookmarked Solutions: Kept until manually deleted
  • User Preferences: Kept until account deletion or extension removal
  • Custom Settings: Synced across devices until changed

6.3 Account Deletion

When you delete your account:

  • All cloud data is permanently deleted within 30 days
  • Local data is cleared from your browser
  • Backups are purged within 90 days
  • Anonymized analytics data may be retained for service improvement

7. Cookies and Local Storage

7.1 Extension Storage

The extension uses Chrome's storage APIs to store:

  • User preferences and settings
  • Solution cache for offline access
  • Recent monitoring results
  • Authentication tokens (if using cloud sync)

7.2 No Tracking Cookies

We do not use cookies for tracking or advertising purposes. Any cookies used are essential for extension functionality.

7.3 Third-Party Services

Our service providers may use their own cookies:

  • Supabase: Authentication and session management
  • Analytics: Anonymized usage statistics (opt-out available)

8. Children's Privacy

Our extension is not intended for users under 13 years of age. We do not knowingly collect personal information from children under 13. If we discover that a child under 13 has provided us with personal information, we will delete it immediately.

Parents and guardians should monitor their children's internet usage and extension installations.
